WeatheryForecast

This project aimed to develop a weather app that includes weather map layers and weather forecast of the chosen city. In order to do this, I used OpenWeather API for weather data and world cities database from SimpleMaps. Weather forecast contains current weather, hourly  forecast for two days, and daily forecast for seven days. Weather map layers include cloud, rain, snow, and temperature layer.

JPPhotography

I have spent several years doing photography, so the object of this project was to build a photography portfolio to showcase my work that also serves as the web store. I implemented login/sign-up functionality so that users can save photos to their wishlist. They also can store their personal information for effortless future purchases.
